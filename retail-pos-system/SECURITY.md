# Security Notes

This repository is a public portfolio/demo version of a retail POS system.

Before deploying for a real business:

1. Do not commit `.env`, `.env.local`, Supabase keys, passwords, or client details.
2. Change all demo login details immediately after setup.
3. Use Supabase Auth or hashed passwords for production use.
4. Review Row Level Security policies before enabling public access.
5. Use demo products and demo shop settings only in the public repository.
6. Keep business names, addresses, phone numbers, customer records, and sales records out of GitHub.

This public version is intended to show features, structure, and development skill. It should not contain private client information.
